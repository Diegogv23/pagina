Descripción

CMS MIG agiliza la creación y administración de sitios y módulos educativos. Permite organizar páginas, gestionar usuarios y roles, y extender funcionalidades mediante componentes configurables, priorizando usabilidad, seguridad básica y buen desempeño.

Características clave

Usuarios y permisos

Roles diferenciados y acceso por secciones.

Módulos y plantillas

Componentes reutilizables para acelerar desarrollo.

Gestor de páginas

Orden, tipos de contenido y navegación clara.

Arquitectura MVC

Código organizado para mantener y escalar.

Buenas prácticas

Estructura limpia y seguridad esencial.

Extensible

Base lista para nuevos módulos sin romper el núcleo.

Galería

Arquitectura y datos

MVC + patrones

MVC con apoyo de Singleton (BD), Factory (componentes) y Observer (eventos).

Modelo de datos

Admin (credenciales, rol, permisos), Pages (páginas y orden), Modules (componentes por página) y Columns (estructura de tablas).

Front-end

HTML/CSS/JS responsive. Base preparada para crecer y optimizar assets.

Preguntas frecuentes

¿Necesita base de datos?

Sí para usuarios, roles y páginas (MySQL/MariaDB). Algunos módulos simples pueden no requerirla.

¿Se puede personalizar?

Sí. La arquitectura modular permite añadir/editar componentes sin tocar el núcleo.

¿Es adaptable a móviles?

El front es responsive; el proyecto puede evolucionar a PWA en el futuro.
